# CampusConnectAI

## Overview

CampusConnectAI is an AI-powered college information assistant developed using Flask, Gemini AI, SQLite, and PDF-based knowledge retrieval. The platform helps students quickly access information related to admissions, fees, placements, courses, and campus resources through an intelligent chatbot.

## Features

* AI-powered chatbot using Google Gemini
* PDF knowledge base integration
* College information management system
* Admin panel for data management
* Student-friendly interface
* Online deployment using Render

## Technologies Used

* Python
* Flask
* SQLite
* Google Gemini AI
* PyPDF2
* HTML
* CSS
* JavaScript
* GitHub
* Render
